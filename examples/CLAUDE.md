# Claude Context Fence

A design pattern for LLM tool skills that separates context awareness
from dense reference material using a router/recipes architecture with
context isolation.

## Key Files

- `README.md` - Pattern overview and quick start
- `SPEC.md` - Full specification (component specs, routing, argument protocol, anti-patterns)
- `examples/` - Reference implementation: 22 skill pairs covering ~195 CLI tools
- `research_docs/` - Archived research, earlier drafts, and test findings
